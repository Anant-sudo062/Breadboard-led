Description

This project demonstrates a simple LED circuit built on a breadboard using a 9V battery, resistor, and LED. It is a beginner-friendly project aimed at understanding the fundamentals of electronics such as current flow, circuit completion, and component connections.

🧰 Components Used
Breadboard
LED (Light Emitting Diode)
Resistor (220Ω – 1kΩ recommended)
9V Battery
Jumper Wires
⚙️ Circuit Explanation

The circuit works by allowing current to flow from the battery through a resistor and then through the LED.

The resistor limits current to protect the LED
The LED glows when current passes through it
Proper polarity is important:
Long leg → Positive (Anode)
Short leg → Negative (Cathode).

Long leg → Positive (Anode)
Short leg → Negative (Cathode)
🔌 How It Works
Connect the positive terminal of the battery to one end of the resistor
Connect the other end of the resistor to the anode of the LED
Connect the cathode of the LED to the negative terminal of the battery
Once the circuit is complete, the LED lights up.
